===============================================
           Tarkov Client v0.1.1 사용법
===============================================

Tarkov Market Pilot을 위한 전용 데스크톱 애플리케이션

[프로그램 소개]
이 프로그램은 Tarkov Market Pilot 웹사이트를 데스크톱 앱으로 실행하며,
게임과 연동하여 실시간 맵 감지 및 위치 추적 기능을 제공합니다.

===============================================
                실행 방법
===============================================

1. TarkovClient.exe 파일을 더블클릭하세요
2. 프로그램이 자동으로 시작됩니다
3. 시스템 트레이(우측 하단)에 아이콘이 표시됩니다

※ 별도 설치 과정이 필요하지 않습니다!

===============================================
            Windows Defender 경고
===============================================

처음 실행 시 보안 경고가 나타날 수 있습니다.
이는 코드 서명 인증서를 구매하지 않았기 때문이며,
프로그램 자체에는 보안 문제가 없습니다.

[해결 방법]
1. "추가 정보" 클릭
2. "실행" 버튼 클릭
3. 정상적으로 프로그램이 시작됩니다

[보안 검증]
※ VirusTotal 검사 완료 (구글 소유의 세계 최고 보안 플랫폼)
※ 70개 이상의 주요 안티바이러스 엔진으로 안전성 확인
※ 검사 결과 확인:
  https://www.virustotal.com/gui/file/982399f3f5d588cb48e4b272b9caf0533202bba86f5cdbb3ed2ad76f621056a4/detection

※ 이 프로그램은 오픈소스로 모든 코드가 공개되어 안전합니다.

===============================================
                기본 사용법
===============================================

[탭 관리]
- 새 탭 추가: 왼쪽 상단의 + 버튼 클릭
- 탭 닫기: 각 탭의 X 버튼 클릭
- 최소 1개 탭은 항상 유지됩니다

[게임 연동]
- 게임을 실행하면 자동으로 맵 감지가 시작됩니다
- 게임에서 스크린샷을 촬영하면 위치가 업데이트됩니다
- 게임 종료 시 자동으로 로그 파일이 정리됩니다

===============================================
               문제 해결
===============================================

[프로그램이 시작되지 않는 경우]
- 관리자 권한으로 실행해보세요
- Windows를 최신 버전으로 업데이트하세요

[웹페이지가 로드되지 않는 경우]  
- 인터넷 연결을 확인하세요
- 방화벽에서 프로그램을 허용하세요

[맵 감지가 되지 않는 경우]
- 게임이 기본 위치에 설치되어 있는지 확인하세요
  (C:\Battlestate Games\Escape from Tarkov\)
- 게임을 관리자 권한으로 실행해보세요

===============================================
              시스템 요구사항
===============================================

- Windows 10 또는 Windows 11 (64비트)
- 메모리: 최소 512MB 여유 공간
- WebView2 Runtime (Windows 11은 기본 포함)

[WebView2 Runtime 설치가 필요한 경우]
Windows 10 사용자는 Windows 업데이트를 실행하세요.
업데이트 후에도 문제가 있다면 Microsoft 웹사이트에서 
'WebView2 Runtime'을 검색하여 다운로드하세요.

===============================================
               주요 기능
===============================================

✓ 실시간 맵 감지 - 게임 로그 기반으로 맵 변경 자동 감지
✓ 위치 추적 - 스크린샷 기반 실시간 위치 및 방향 업데이트
✓ 다중 탭 지원 - 여러 웹페이지 동시 사용 가능
✓ 자동 파일 정리 - 로그 폴더 및 스크린샷 자동 정리 (성능 최적화)
✓ 시스템 트레이 - 백그라운드에서 조용히 실행
